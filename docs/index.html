Build Instructions

Follow these steps to set up the project locally:

  1. Clone the repository git clone https://github.com/alexproc03/AlLogic.git or git clone git@github.com:alexproc03/AlLogic.git
  2. Navigate to the project directory
  3. Install dependencies: pip install textX
  4. Run the project via terminal with al file as target python AlLogic.py (.al file)
